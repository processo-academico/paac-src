Work Product Descriptor (Artifact): Plano de Desenvolvimento de Software
O Plano de Desenvolvimento de Software define as atividades do desenvolvimento dentro das fases e iterações necessárias para a implementação do projeto. Ele descreve o plano de desenvolvimento que cobre o projeto como um todo.
